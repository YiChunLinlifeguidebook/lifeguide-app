# 生命指南 - Life Guide App

一個完全中文化的生命指導網站應用程式，旨在為使用者提供全方位的人生規劃與發展指引。

## 功能特色

- ✨ **完整中文介面** - 所有介面元素均已中文化
- 🎨 **現代化設計** - 採用響應式設計，支援各種裝置
- 📱 **行動裝置友善** - 完美適配手機和平板電腦
- 🚀 **快速載入** - 輕量級設計，無需複雜依賴

## 主要內容

### 首頁
歡迎使用者並介紹網站的核心價值

### 關於我們
- 專業諮詢服務
- 豐富的學習資源
- 支持性社群環境

### 服務項目
1. 個人發展計劃
2. 職涯規劃諮詢
3. 生活平衡指導
4. 心靈成長課程

### 資源中心
- 文章與部落格
- 影音課程
- 可下載資源
- 常見問題解答

### 聯絡我們
提供聯絡表單讓使用者可以輕鬆與我們溝通

## 安裝與使用

### 直接開啟
1. 克隆此儲存庫：
```bash
git clone https://github.com/YiChunLinlifeguidebook/lifeguide-app.git
cd lifeguide-app
```

2. 直接用瀏覽器開啟 `index.html` 檔案

### 使用本地伺服器（推薦）
1. 安裝依賴：
```bash
npm install
```

2. 啟動開發伺服器：
```bash
npm start
```

3. 瀏覽器會自動開啟 http://localhost:8080

## 技術架構

- **HTML5** - 語義化標記
- **CSS3** - 現代化樣式與動畫效果
- **JavaScript (ES6+)** - 互動功能實作
- **響應式設計** - 適配各種螢幕尺寸

## 中文化細節

所有介面元素均使用繁體中文：
- 導航選單
- 標題與內容
- 按鈕文字
- 表單標籤與提示
- 版權資訊

字型選擇：
- 主要字型：Microsoft YaHei（微軟雅黑）
- 備用字型：PingFang TC（蘋方繁體中文）
- 英文字型：Helvetica Neue, Arial
